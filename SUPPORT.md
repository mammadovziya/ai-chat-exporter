# Support

For provider breakage, open a **Provider breakage** issue and include:

- provider
- browser
- extension version
- popup diagnostics from **Check**
- screenshot with private content redacted

For security issues, do not open a public issue. Use GitHub private security
advisories or follow `SECURITY.md`.

The project cannot recover chats that were not loaded in the browser. If older
messages are missing, scroll the chat until they load, then press **Refresh** in
the export panel.
